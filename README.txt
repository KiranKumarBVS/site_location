INTRODUCTION
------------

This module will display the Site location and the current time for the selected timezone location.

This module provides the below facilities:
  * It provides a configuration setting at (admin/config/system/site-location) to configure .
  * Data can be seen in the "Site location block".


REQUIREMENTS
------------

No requirements!

This module requires no modules outside of Drupal core.


INSTALLATION
------------

 * Enable the module by using one of the methods.
    1. By using drush command: "drush en site_location"
    2. By visiting the (/admin/modules), select the "Site location" module and click on install.


CONFIGURATION
-------------

 * Goto: (admin/config/system/site-location).
   One settings config form will be there.

 * Set your configuration

 * Click on the save button to save the configuration.

 * Goto: (/admin/structure/block).
   Place the block in the preferred region.

 * Click on Save blocks.
